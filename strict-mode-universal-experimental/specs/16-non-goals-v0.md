# 16. Non-Goals For v0

Part of [Strict Mode Universal Experimental - Specification v0](../SPEC.md).


- Transcript-scraping parity for FDR challenge before fixture-proven bounded current-turn extraction
- Full interactive `/fdr` runner implementation beyond required `strict-fdr import`
- CI integration
- Windows support
- GUI installer
- Replacing existing Claude-only bundle immediately
- Auto-migrating all historical `~/.claude/state` files
- Treating cheap worker-model output as authoritative review, approval, or Stop allow evidence
- Sending whole repositories, raw transcripts, or hidden provider history to workers for token savings
